# Legal Source Policy

이 프로젝트는 법적 리스크를 줄이기 위해 다음 원칙으로 수집합니다.

## 원칙
- 로그인/우회/헤드리스 브라우저를 이용한 강제 크롤링 금지
- robots.txt 및 서비스 약관에 반하는 수집 금지
- 공개 RSS/공식 API/공개 목록 페이지처럼 접근 제한이 없는 채널만 사용
- 원문 전문 저장 최소화, 목록에서 확인 가능한 제목/메타 중심으로 요약
- 출처(source), 플랫폼(platform), 원문 URL을 항상 표시
- 사이트 내부 상세 보기에서는 가공 요약만 보여주고 원문 커뮤니티 글로 연결

## 2026-03-30 운영 상태
- 활성 수집: 뽐뿌, 해외뽐뿌, 알리뽐뿌, FM코리아, 개드립, 루리웹, 딜바다 국내, 딜바다 해외, 쿨엔조이
- 비활성 수집: 퀘이사존(`HTTP 403`), 아카라이브(`Cloudflare challenge 403`), 클리앙(`HTTP 403`)

## 운영 체크리스트
1. 소스 추가 전 이용약관 확인
2. robots/정책에 자동 수집 허용 여부 확인
3. 요청 빈도 제한(레이트 리밋) 적용
4. 저작권/상표권 문제 소지 있으면 소스 제외

## 설정 파일
- 소스 허용 목록: `config/sources.json`
- 수집 스크립트는 위 파일에 등록된 공개 RSS/목록 소스만 처리
